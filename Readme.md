# Student Grade Calculator

## Project Description
A Python-based student grade calculator that processes multiple students, calculates grades with personalized comments, and displays class statistics.

## What I Learned
- Conditional logic using if/elif/else
- Lists and list comprehensions
- For and while loops
- Error handling using try-except
- Modular programming using functions
- File handling in Python

## Features
- Multiple student processing
- Grade calculation with comments
- Input validation
- Search functionality
- Save results to file
- Menu-driven system
